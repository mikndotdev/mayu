# 見出し1（H1）

これは見出し1のサンプルテキストです。最も大きな見出しとして使用されます。

## 見出し2（H2）

見出し2は主要なセクションの区切りとして使用されます。

### 見出し3（H3）

見出し3はサブセクションに使用されます。

#### 見出し4（H4）

見出し4はさらに細かい区分に使用されます。

##### 見出し5（H5）

見出し5は階層が深い場合に使用されます。

###### 見出し6（H6）

見出し6は最も小さな見出しです。

## テキストの装飾

これは通常の段落テキストです。段落は自動的に適切な行間と余白でレンダリングされます。

**太字テキスト**や*イタリック体テキスト*、***太字とイタリックの組み合わせ***も使用できます。

また、`インラインコード`を使用して、短いコードスニペットやコマンドを強調できます。

## リンク

[これはリンクです](https://astro.build)。外部サイトへのリンクとして機能します。

[内部リンクの例](/components)は、サイト内の他のページにリンクできます。

## リスト

### 順序なしリスト

- アイテム1
- アイテム2
- アイテム3
  - ネストされたアイテム1
  - ネストされたアイテム2
    - さらにネストされたアイテム
- アイテム4

### 順序ありリスト

1. 最初のステップ
2. 2番目のステップ
3. 3番目のステップ
   1. サブステップA
   2. サブステップB
4. 最後のステップ

### チェックリスト

- [x] 完了したタスク
- [x] もう一つの完了したタスク
- [ ] 未完了のタスク
- [ ] 別の未完了のタスク

## コードブロック

### JavaScript

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
  return `Welcome to our site, ${name}`;
}

const user = "太郎";
greet(user);
```

### TypeScript

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

const createUser = (name: string, email: string): User => {
  return {
    id: Math.random(),
    name,
    email,
  };
};
```

### HTML

```html
<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>サンプルページ</title>
  </head>
  <body>

```

## 引用

> これは引用ブロックです。他の人の言葉や重要な情報を強調するために使用されます。
>
> 複数行にわたる引用も可能です。

> **注意:** 引用ブロック内で**太字**や*イタリック*などの装飾も使用できます。
